数据结构训练一：字符串置换

FROM lintcode MARK 数组、字符串

题目

给定两个字符串，请设计一个方法来判定其中一个字符串是否为另一个字符串的置换。

置换的意思是，通过改变顺序可以使得两个字符串相等。

输出样例
“abc” 为 “cba” 的置换。

“aabc” 不是 “abcc” 的置换。

> 先放代码

public class Solution {
/**

 * @param A a string
 * @param B a string
 * @return a boolean
 */
public boolean stringPermutation(String A, String B) {
    // Write your code here
    if(A.length()!=B.length()) {
        //判断长度是否一致，否返回false
        return false;
    } else {
        //进行字符串的分割
        String[] arrA = A.split("");
        String[] arrB = B.split("");
        //用一个数组来保存比较结果
        int[] mark = new int[arrA.length];
        for(int i=0; i<mark.length; i++) {
            //数组初始化为全0
            mark[i] = 0;
        }
        for(int i=0; i<arrA.length; i++) {
            for(int j=0; j<arrB.length; j++) {
                if(arrB[j].equals(arrA[i]) && mark[j]==0) {
                    //若字符串一致并且int数组该index下值为0，则置为1
                    mark[j] = 1;
                    break;
                }
            }
        }
        for(int i=0; i<mark.length; i++) {
            if(mark[i]==0) {
                //最后遍历数组，出现0则直接返回false
                return false;
            }
        }
        //全部为1返回true
        return true;
    }
}

}

我的思路

  • 判断字符串长度是否相同，不相同直接返回false
  • 长度相同的情况下，将获取到的两个字符串进行分割，用split(“”)，返回字符串数组
  • 通过嵌套循环比较两个字符串数组中的每个字符串是否相同；比较的结果通过一个与字符串数组长度相同的int数组来保存（相同设置为1）；循环结束后，判断int数组是否全部为1，是则返回true，否则返回false

结果

  • 代码可以完成预定功能。
    思考
  • 嵌套循环的使用使得程序效率低下，时间、空间复杂度都较高
